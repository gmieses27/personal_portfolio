Arbiter — Authorization Gateway

HTTP reverse proxy that enforces RBAC policies before requests hit backends. Resolves caller identity from JWT or API key, evaluates rules via a thread-safe policy engine (deny overrides allow), and proxies or rejects. Hot-reloads policies on SIGHUP — no restart needed.

Go RBAC / AuthZ Reverse Proxy SSE JWT
→ Live demo → GitHub
★ FEATURED — Live Demo in Demos Section
Distributed Key-Value Store

Fault-tolerant distributed database built from scratch in Go, implementing the Raft consensus algorithm. Three nodes coordinate over HTTP to elect a leader, replicate every write through a shared log, and recover automatically after crashes or network partitions. Includes a real-time cluster dashboard with live role visualization and interactive partition controls.

Go Raft Consensus Distributed Systems SSE HTTP
→ Live demo → GitHub

Procedural Plant Generation Engine

Parametric L-system engine in C++ generating complex 2D/3D plant geometries. Built a 3D turtle graphics interpreter with quaternion rotations, 8-way branching, and tropism simulation. Real-time OpenGL rendering at 60 FPS with 90K+ vertices per plant and live parameter tuning via ImGui.
